Welcome to Blocks! This tool lets you explore and interact with generative artworks from Art Blocks and other collections directly from Ethereum.

Getting Started

  • First, you must provide a RPC URL for the Ethereum network. Alchemy
  • Paste your RPC URL below and press enter

How to Use

  • Search Box Commands:
    • Leave empty + enter: View random artwork
    • Type collection name + enter: Random piece from that collection
    • Type "collection #123" + enter: View specific artwork
    • Type just "#123" + enter: View specific artwork from current collection
  • Use \ to reset the page
  • Click artwork info to see details
  • Save your favorites with the disk icon
  • Use loop features to auto-explore collections

Support Decentralization

Help keep this tool decentralized by pinning it to your IPFS node
